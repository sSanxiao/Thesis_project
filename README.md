# Thesis_project
## Study of Cell Spatial Density in Cell Development and Cancer Oncogenesis
This project inculdes four Stages according to Prof. Feng Zhang's design:

Stage 1. Data Collection (~1 month): Collect currently available downloadable Xenium datasets (search paper in Pubmed or Google Scholar). The target is to focus on those related to the development and cancer generation in the brain or cerebellum.

Stage 2. Data Analysis (~1 month): Go through the basic analysis (basic scripts are provided by Prof. Feng Zhang)

Stage 3. Further Analysis (depends): Identify "spatial density" related genes. Perform integrated analysis (statistics or machine learning) of these genes with single-cell or bulk data related to cell development and cancerigenesis, to explore the relationships between these genes and the two processes.

Stage 4. Result Summary (depends): Summarize the above findings, three tiers of conclusions can be drawn based on the results:

I. Strong correlation: If a strong correlation is observed, predictive models for cell development or cancerigenesis can be established and  validated in datasets such as TCGA.

II. Moderate correlation: If the correlation is moderate, model construction may not be suitable, but summary descriptions can be provided to elaborate on the identified patterns (e.g., gene pathways) in detail.

III. Weak or no correlation: In this case, only a few genes can be used as examples, and the functions of these identified genes can be thoroughly explained through a literature review.
